# SGD 為何收斂到好解？其普遍成立的充分條件是什麼？

## A. 最終立場：部分成立，且有精確邊界

SGD 收斂到好解**不是**無條件的普遍事實。存在四類嚴格反例（F1–F4）使其失敗。但在失敗條件均不成立時，SGD 的隱式正則化機制可以被完整證明，且在兩個機制域中各有一個閉環定理。

**不存在單一的普遍充分條件**，但存在一個統一的選擇原理：SGD 的隨機性充當 per-sample 靈敏度的篩選器，在不同機制域中以不同數學形式體現。

最終答案：

$$\text{SGD 收斂到好解} \;\Longleftrightarrow\; \neg F1 \wedge \neg F2 \wedge \neg F3 \wedge \neg F4$$

即：
1. 學習率夠大使盆地間混合（¬F1）
2. 學習率不至於太大使 SGD 發散（¬F2）
3. 好解與壞解的 per-sample 靈敏度有差異（¬F3）
4. 低靈敏度確實對應好泛化（¬F4）

前三個是演算法的條件，第四個是問題本身的條件——數據的結構必須使得「簡單函數」（低 Jacobian 範數）恰好是「正確函數」。

## B. 形式模型與符號

### B.1 基本定義

**SGD**：給定參數空間 Θ ⊆ ℝ^d，損失函數 L(θ) = (1/n)Σᵢ ℓᵢ(θ)，SGD 迭代為：

θ_{t+1} = θ_t - η ∇ℓ_{z_t}(θ_t),  z_t ~ Uniform({1,...,n}) iid

**梯度噪聲方差**：

Σ(θ) = (1/n)Σᵢ [ℓ'ᵢ(θ)]² - [L'(θ)]²

在極小值 θ* 處 L'(θ*) = 0，故 Σ(θ*) = (1/n)Σᵢ [ℓ'ᵢ(θ*)]²。

**收斂**（三層定義）：
- (C1) 優化收斂：L(θ_t) → L* a.s.
- (C2) 分佈收斂：θ_t →^d μ_η（某穩態測度）
- (C3) 吸收收斂：θ_t → θ* a.s.

**好解**（本文採用 reparameterization-invariant 定義）：
- (G1) 低噪聲比：Σ(θ*)/H(θ*) 在所有極小值中較小
- (G2) 低 Jacobian 複雜度：C(θ*) = ||J||_F/√n 小（插值域）
- (G3) 穩定解：leave-one-out 穩定性 β 小
- (G4) 泛化解：L_test - L_train ≤ ε 小

核心關聯（本文證明）：SGD 偏好 (G1)/(G2)，而 (G1)→(G3)→(G4)，(G2)→(G4)。

### B.2 連續時間 SDE 近似（非插值域）

dθ = -∇L(θ) dt + √(τ Σ(θ)) dW_t,  τ = η/B

### B.3 離散線性化（插值域）

在插值解 θ* 附近，MSE loss 下 ∇²ℓᵢ(θ*) = JᵢJᵢᵀ：

δ_{t+1} = (I - η J_{z_t} J_{z_t}ᵀ) δ_t

其中 Jᵢ = ∇_θ f_{θ*}(xᵢ)，hᵢ = ||Jᵢ||²。

## C. 核心定理

### 定理 I：非插值域閉環（1D，完整證明）

**假設**：
- (A1) L, ℓᵢ ∈ C²(ℝ)
- (A2) L coercive：L(θ) → +∞ 當 |θ| → ∞
- (A3) Σ(θ) 嚴格正且有界：0 < σ²_min ≤ Σ(θ) ≤ σ²_max
- (A4) L 恰好有兩個等高局部極小值 θ_A, θ_B，H_A, H_B > 0
- (A5) Σ_A < Σ_B（對稱情形；一般情形需 A5'）
- (A6) Σ_A/H_A < Σ_B/H_B

**結論**（三部分，每步逐行證明）：

**Part I（動力學）**：SGD-SDE 穩態分佈存在且唯一：

p(θ) = (1/Z) · (1/Σ(θ)) · exp(-(2/τ) ∫^θ L'(s)/Σ(s) ds)

等價地 p(θ) ∝ exp(-(2/τ) V_eff(θ))，其中有效勢能：

V_eff(θ) = ∫^θ L'(s)/Σ(s) ds + (τ/2) ln Σ(θ)

**Part I 證明**：

1. FPE（Itô）：∂p/∂t = ∂/∂θ[L'p] + (τ/2) ∂²/∂θ²[Σp]
2. 穩態 ∂p/∂t = 0，定義機率流 J = -L'p - (τ/2)(Σp)'
3. dJ/dθ = 0 → J = const
4. 邊界條件（A2+A3）：p → 0 當 |θ| → ∞ → J ≡ 0
5. 展開 J = 0：L'p + (τ/2)[Σ'p + Σp'] = 0
6. p'/p = -2L'/(τΣ) - Σ'/Σ
7. 右側 = -d/dθ[(2/τ)∫ L'/Σ dθ + ln Σ]（微積分基本定理 + (ln Σ)' = Σ'/Σ）
8. 積分：ln p = -(2/τ)∫ L'/Σ dθ - ln Σ + C
9. 取指數得結論
10. 歸一化 Z < ∞ 由 (A2)(A3) 保證 ∎

**Part II（選擇）**：在對稱情形下（L 和 Σ 關於 (θ_A+θ_B)/2 對稱），

p(θ_A)/p(θ_B) = Σ_B/Σ_A > 1

**Part II 證明**：

1. 由 Part I，p(θ_k) ∝ (1/Σ_k) exp(-(2/τ) ∫^{θ_k} L'/Σ dθ)
2. 取比值，(1/Σ) 因子比為 Σ_B/Σ_A
3. 指數差 = (2/τ) ∫_{θ_A}^{θ_B} L'/Σ dθ
4. 對稱情形下 L'/Σ 是反對稱函數，在對稱區間積分 = 0
5. 故 p(θ_A)/p(θ_B) = Σ_B/Σ_A > 1（由 A5）∎

**Part III（泛化）**：Gap(θ*) ≈ Σ(θ*)/(H(θ*)(n-1))

**Part III 證明**：

1. θ* 是 L 的極小值，L'(θ*) = 0
2. 移除第 j 個樣本：L'_{-j}(θ*) = -ℓ'_j(θ*)/(n-1)
3. L''_{-j}(θ*) = (nH - ℓ''_j)/(n-1) ≡ H_{-j}
4. 隱函數定理：θ*_{-j} - θ* ≈ ℓ'_j(θ*)/((n-1)H_{-j})
5. LOO 損失差：ℓ_j(θ*_{-j}) - ℓ_j(θ*) ≈ [ℓ'_j(θ*)]²/((n-1)H_{-j})
6. 平均：(1/n)Σ_j [ℓ'_j]²/((n-1)H_{-j}) ≈ Σ(θ*)/((n-1)H)
7. 識別：(1/n)Σ_j [ℓ'_j(θ*)]² = Σ(θ*)（因 L'(θ*) = 0）
8. 泛化間隙 ≈ L_test - L_train ≈ Σ/(H(n-1))
9. 由 (A6)：Gap(θ_A) = Σ_A/(H_A(n-1)) < Σ_B/(H_B(n-1)) = Gap(θ_B) ∎

**綜合**：SGD 偏好 θ_A（Part II），θ_A 泛化更好（Part III）。∎

### 定理 II：插值域閉環（離散，完整證明）

完整 LaTeX 證明見 `sgd_interpolation_theorem_final.tex`。

**假設**：
- (A1) f_θ(x) 對 θ 二次連續可微，||∇²_θ f(x_i)||_op ≤ B₂
- (A2) {J₁,...,Jₙ} 線性獨立
- (A3) Jᵢᵀ Jⱼ = 0 對 i ≠ j（Jacobian 正交性）

**定義**：
- 臨界學習率：η_c(θ*) = 2/h_max(θ*)
- η-穩定（好的）：η h_max < 2
- η-不穩定（壞的）：η h_max > 2
- Jacobian 複雜度：C(θ*) = ||J||_F/√n = √((1/n)Σᵢ hᵢ)

**Part A（壞解被排斥）**：

若 η h_max > 2，則最大 Lyapunov 指數 λ = max_k (1/n)ln|1-ηh_k| > 0，SGD 幾乎必然逃離。

**證明要點**：
1. 正交假設下，A_i = I - ηJ_iJ_iᵀ 在正交基 {J_k/√h_k} 下完全解耦
2. 每個坐標獨立演化：(ξ_{t+1})_k = (1-ηh_k)(ξ_t)_k（當 z_t = k）或 (ξ_t)_k（當 z_t ≠ k）
3. ln|(ξ_T)_k| = ln|(ξ_0)_k| + Σ_t X_t^{(k)}，X_t^{(k)} iid
4. 強大數定律：(1/T)Σ X_t → λ_k = (1/n)ln|1-ηh_k|
5. η h_{k*} > 2 → |1-ηh_{k*}| > 1 → λ_{k*} > 0 → |ξ_T| → ∞ a.s. ∎

**Part B（好解穩定）**：

若 η h_max < 2，則所有 λ_k < 0，活躍子空間 S 上的偏差指數衰減。

**證明**：同 Part A 但符號相反——η h_k < 2 → |1-ηh_k| < 1 → λ_k < 0 → |ξ_T| → 0 a.s. ∎

**Part C（好解泛化更好）**：

策略：leave-one-out 在插值域失效（∇ℓ_j(θ*) = 0），改用 Rademacher 複雜度。

引理（Jacobian 複雜度控制 Rademacher）：定義 G_ρ = {x ↦ f_θ(x) - f_{θ*}(x) : ||θ-θ*|| ≤ ρ}，

R̂_S(G_ρ) ≤ ρ C(θ*)/√n + B₂ρ²/2

**證明**：
1. Taylor：f_θ(x_i) - f_{θ*}(x_i) = Jᵢᵀδ + r_i(δ)，|r_i| ≤ B₂ρ²/2
2. Rademacher 分解：R̂ ≤ E_σ[sup_{||δ||≤ρ} (1/n)Σ σ_i J_iᵀδ] + B₂ρ²/2
3. 線性部分：sup = (ρ/n)||Σ σ_i J_i||
4. Jensen：E[||Σ σ_i J_i||] ≤ √(E[||Σ σ_i J_i||²]) = √(Σ h_i) = √n · C
5. 合併：R̂ ≤ ρC/√n + B₂ρ²/2 ∎

泛化界：由 Ledoux-Talagrand 收縮 + 標準 Rademacher 泛化界，

L_pop(θ*) ≤ 0 + 2L_ℓ ρC/√n + L_ℓ B₂ρ² + ō√(ln(2/δ)/(2n))

SGD 選出的解：C² ≤ h_max < 2/η，故主項 ≤ O(ρ/√(ηn))。∎

**主定理（閉環）**：

(I) SGD 排斥 M\M_η（h_max > 2/η 的解），吸引 M_η（h_max < 2/η 的解）
(II) M_η 中的解滿足 C ≤ √(2/η)
(III) 泛化界 L_pop ≤ O(ρ/√(ηn))
(IV) η 增大 → M_η 縮小 → C 上界下降 → 泛化界收緊

## D. 四類反例與失敗邊界

### 反例 F1：η 太小（動力學陷阱）

**構造**：雙井勢，壁壘 ΔV = 100，井 A 好（H=2, Σ=1），井 B 壞（H=8, Σ=4），初始化在 B。

**行為**：Kramers 逃逸時間 τ_{B→A} ~ (1/η)exp(12.5/η)。η = 0.01 時 τ ~ 10^545。SGD 被鎖定在壞解 B。

**失敗原因**：混合時間假設被違反。SGD 未達穩態。

### 反例 F2：η 太大（發散/混沌）

**構造**：L = (1/2)·10·θ²，兩個樣本的 Hessian 分別為 18 和 2。η = 0.15。

**行為**：Lyapunov 指數 Λ = (1/2)(ln 1.7 + ln 0.7) = 0.087 > 0。|θ_t| ~ e^{0.087t} → ∞。

**失敗原因**：η > 2/λ_max = 2/18 ≈ 0.111。離散穩定性破壞。

### 反例 F3：無差異化（選擇力退化）

**構造**：兩井 A, B，刻意設 Σ_A/H_A = Σ_B/H_B = 1。A 泛化好，B 泛化差。

**行為**：有效勢在兩井的曲率相同 → SGD 穩態 π(A) ≈ π(B) ≈ 1/2。50% 概率停在壞解。

**失敗原因**：Σ/H 在極小值間無差異，噪聲偏好機制完全失效。

### 反例 F4：噪聲-泛化反轉（最深刻）

**構造**：好解 A（泛化好但 Σ_A = 20 高），壞解 B（泛化差但 Σ_B = 1 低）。等損失 L_A = L_B = 0.1，等曲率 H_A = H_B = 2。

**行為**：有效壁壘 ΔU_A = 0.5/η（低 → 易逃逸），ΔU_B = 9.8/η（高 → 難逃逸）。τ_A/τ_B ~ exp(-9.31/η) → 0。SGD 以壓倒性概率停在壞解 B。

**失敗原因**：好解用簡單特徵（高數據多樣性 → 高 Σ），壞解記憶噪聲（各樣本梯度一致 → 低 Σ）。SGD 的低 Σ 偏好與泛化需求衝突。

**F4 是主定理的根本邊界**：在過參數化插值域中，過擬合解的梯度噪聲天然很低（Σ ≈ 0），這是 Σ 偏好與泛化可能衝突的深層原因。

### 最小失敗條件

SGD 收斂到好解 ⟺ 以下四個條件同時成立：

1. ηT ≫ τ_mix（充分混合，¬F1）
2. η < 2/λ_max（穩定，¬F2）
3. Σᵢ/Hᵢ 在極小值間有差異（有選擇力，¬F3）
4. Corr(Σ, 泛化間隙) > 0（方向正確，¬F4）

## E. 概念問題的嚴格回答

### E1. Simplicity vs Flatness

**結論**：Σ/H 是 reparameterization-invariant 的，H 和 Σ 單獨都不是。

**證明**：設 θ = g(ϕ) 是光滑雙射，ϕ* = g⁻¹(θ*)。

H̃ = L''(g(ϕ*)) · [g'(ϕ*)]² = H · (g')²

Σ̃ = E_i[(ℓ'_i(g(ϕ*)))²] · [g'(ϕ*)]² = Σ · (g')²

故 Σ̃/H̃ = Σ·(g')² / (H·(g')²) = Σ/H ∎

**推論**：
- Flatness（H 小）可被 reparameterization 任意改變而不影響泛化 → 單獨不保證泛化
- Simplicity（Σ 小）同理
- 但 Σ/H 不變 → 泛化間隙 ≈ Σ/(H(n-1)) 是內在幾何量

**多維推廣**：tr(H⁻¹Σ) 也是不變的（Jacobian 的 (Jᵀ)² 在分子分母抵消）。

### E2. 最小範數投影

**定理**：SGD 從 w₀ = 0 出發，在過參數化線性模型 f(x;w) = wᵀx 中收斂到最小 ℓ₂ 範數插值解 w* = X†y = Xᵀ(XXᵀ)⁻¹y，與 GD 相同。

**證明**：
1. w₀ = 0 ∈ row(X)
2. SGD 增量 = -η(wᵀx_{z_t} - y_{z_t})x_{z_t} ∈ row(X)
3. 歸納：w_t ∈ row(X) ∀t
4. row(X) 中唯一的插值解是 X†y
5. 凸損失上 SGD 收斂定理 → w_t → X†y ∎

**意義**：這是代數性質，與噪聲無關。在非線性模型中 SGD 與 GD 的隱式偏差可能不同。

### E3. 多維 Detailed Balance

**成立條件**：curl(Σ⁻¹∇L) = 0，即 ∂_i[Σ⁻¹∇L]_j = ∂_j[Σ⁻¹∇L]_i ∀i≠j。

**特殊情況**：
- Σ = σ²(θ)I 且 σ² = h(L) → 自動保守
- Σ 常數且 [Σ⁻¹, H] = 0 → 保守
- 一維 → 自動成立（無橫向分量）

**破缺時**：穩態仍為 p ~ exp(-V/τ)，但 V 是 Freidlin-Wentzell 準勢，需解 Hamilton-Jacobi 方程 -∇L·∇V + (1/2)(∇V)ᵀΣ∇V = 0。旋轉流不改變指數項的形式，但改變 V 的值。

**2D 具體例子**：∇L = (θ₁, θ₂)，Σ = diag(1+θ₂², 1)。

Detailed balance 破缺：∂/∂θ₂[θ₁/(1+θ₂²)] = -2θ₁θ₂/(1+θ₂²)² ≠ 0 = ∂/∂θ₁[θ₂]。

準勢（Hamilton-Jacobi 求解）：V = θ₁² + θ₂² - (1/2)θ₁²θ₂² + O(|θ|⁶)。

修正項 -(1/2)θ₁²θ₂² 使得噪聲大的方向（θ₁）的準勢降低 → SGD 在該方向花更多時間。

### E4. 三層條件分類

**必要條件**：V_eff 的全局最小值落在好解集 G 中。
- 證明：穩態在 τ → 0 時集中在 V_eff 全局最小值上。若最小值不在 G 中 → 穩態集中在壞解 → 矛盾。

**充分條件**：V_eff 全局最小唯一在 G 中，能隙 Δ > 0，τ ≪ Δ。
- 結論：P(好解) ≥ 1 - (K-1)exp(-2Δ/τ)

**近似充分條件**：所有 V_eff 值 ≤ V_eff^min + δ 的極小值都近似好（Gap ≤ ε + ε'）。
- 結論：P(Gap ≤ ε + ε') ≥ 1 - γ，其中 γ ~ Ke^{-2δ/τ}

### E5. 各因素的嚴格影響

**η 增大**：
- 非插值域：離散修正 V_eff^(η) = V_eff - (η/2)ln Σ → 削弱噪聲選擇力（-ln Σ 項推向高 Σ）
- 插值域：M_η = {θ*: h_max < 2/η} 縮小 → 篩選更嚴 → C 上界下降 → 泛化界收緊
- 極小值偏移：δθ ≈ ηΣ'/(2H)

**B 增大**：
- τ = η/B 減小 → 分佈更集中在 V_eff 最小值
- V_eff 形狀不變 → 極小值排序不變
- 波動幅度 ~ √(η/B) 隨 B 增大而減小

**過參數化（d > n）**：
- 插值流形 M 變成高維子流形
- M 上所有點 Σ = 0 → SDE 退化 → 轉為離散分析
- SGD 在 M 上的有效漂移 v_eff = -(τ/2)∇_∥ ln det Σ_⊥ → 漂移到法方向噪聲最小的位置

**曲率（Hessian 譜）**：
- 泛化間隙的多維版：tr(H⁻¹Σ) = Σᵢ σ²ᵢ/λᵢ
- 平坦方向（小 λᵢ）主導泛化間隙
- 有效維度 d_eff = (tr H)²/tr(H²) 可遠小於 d

**對稱性**：
- V_eff(g·θ) = V_eff(θ) 對 g ∈ 對稱群 G（函數不變性）
- 物理解的有效權重：p_phys([θ]) = |G/G_θ| · p(θ)（軌道計數）
- 穩定化子小的解（高對稱破缺）得到更大權重
- 效應量級：O(τ)，在 τ → 0 時可忽略

## F. 噪聲的四重角色

| 角色 | 機制域 | 數學機制 | 來源 |
|------|--------|---------|------|
| **探索** | 非插值 | Kramers 逃逸 ~exp(-ΔV/τ) | 連續+離散 |
| **選擇** | 非插值 | V_eff ≠ L，穩態偏好低 Σ | 連續（乘性噪聲 SDE） |
| **篩選** | 插值 | h_max > 2/η → Lyapunov 指數 > 0 | **純離散** |
| **鎖定** | 插值 | ∇ℓ_i = 0 → 精確不動點 | **純離散** |

核心洞見：在插值域（現代深度學習的主要工作域）中，SGD 的正則化完全是離散效應，與連續 SDE 近似無關。

---

## G. 可驗證預測

1. **h_max 預測泛化**：訓練至插值後測量 h_max = max_i ||∇_θ f(x_i)||²。不同 η 訓練的模型，h_max 應與測試誤差正相關，相關性強於 Hessian 跡或 flatness。

2. **臨界相變**：固定模型在 η ≈ 2/h_max 處存在尖銳相變——訓練損失從 0 跳到非零。

3. **噪聲結構 vs 量級**：將 SGD 噪聲替換為等量級各向同性高斯，泛化應下降。

4. **η 單調效應**：插值域中 η 增大 → C = ||J||_F/√n 單調下降（直到 SGD 無法收斂）。

5. **F4 可觀測性**：小數據大模型設定中，過擬合解的 Σ 低於泛化解的 Σ。

## H. 未解決的關口

1. **多維 V_eff**：一維 detailed balance 自動成立。多維需 Freidlin-Wentzell 準勢（NP-hard 變分問題）。
